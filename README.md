# 🌌 ZodiacWeb

Projeto Full Stack sobre astrologia, signos e horóscopo.

## 🚀 Funcionalidades

- Cadastro de usuários
- Login seguro com bcrypt
- Quiz zodiacal
- Compatibilidade entre signos
- Horóscopo
- Música ambiente
- Tema astral animado

## 🛠 Tecnologias

### Frontend
- HTML
- CSS
- JavaScript

### Backend
- Node.js
- Express
- MySQL

### Deploy
- Railway
- Netlify

## 🔐 Segurança

- Senhas criptografadas com bcrypt
